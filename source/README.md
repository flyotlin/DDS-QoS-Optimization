# Run the experiment
We hope to prepare a uniform src code for the 3 cases.

So that you only need to provide configurations for 3 cases.

## Configs
`configs.yaml` 中能指定所有本次實驗要調整的參數，參數包含：
- repoUrl
- fastddsPath
- repoPath
- configs: 所有需要使用的
- sleep_time
- trial_num
- case_num
- total_messages
- transport
- sending_rate
- host{1..4}

## `common_vars.yaml`
- logPath
  - generated by `generate.py`
- basePath
  - generated by `generate.py`
- repoUrl
- repoPath
